# Phishing Simulation Explanation

## Overview

A controlled phishing simulation was created to demonstrate how attackers use social engineering techniques to trick users.

## Components

### 1. Fake Email Scenario

A simulated phishing email was created using urgency-based language to mimic real-world attacks.

### 2. Fake Login Page

A replica login page was designed to demonstrate credential harvesting techniques.

### 3. User Behavior Analysis

The simulation demonstrates how users interact with suspicious links and deceptive interfaces.

## Attacker Techniques Demonstrated

- Email spoofing

- Social engineering (fear & urgency)

- Fake domain imitation

- Credential harvesting

## Security Note

This simulation was conducted in a safe and controlled environment for educational purposes only.
