# CodeAlpha_Phishing_Awareness_Training

## Overview

This project is a cybersecurity awareness training module focused on phishing attacks, fake websites, and social engineering techniques.

## Objective

To educate users on how phishing attacks work and how to identify and prevent them.

---

## Phishing Simulation

A controlled simulation was created to demonstrate attacker techniques such as:

- Fake phishing emails

- Fake login pages

- Social engineering manipulation

---

## Topics Covered

- Phishing attacks

- Fake websites

- Social engineering tactics

- Email analysis

- Cybersecurity best practices

---

## Screenshots

- Phishing email example

- Fake login page demonstration

---

## Quiz Section

An interactive quiz was included to test user understanding of phishing awareness.

## Tools Used

-  Canva

## Key Learning Outcomes

- Understanding phishing attack structure

- Identifying fake emails and websites

- Recognizing social engineering tactics

- Improving cybersecurity awareness
