# Phishing Awareness Quiz

## Q1

What is a common phishing tactic?

A. Software updates  

B. Urgent account warning emails  

C. Firewall alerts  

D. System backup  

Answer: B

---

## Q2

Which is a sign of a fake website?

A. HTTPS secure lock  

B. Correct domain name  

C. Misspelled domain name  

D. Verified certificate  

Answer: C

---

## Q3

What should you do if you receive a suspicious email?

A. Click immediately  

B. Ignore security warnings  

C. Verify sender before clicking  

D. Download attachment  

Answer: C
